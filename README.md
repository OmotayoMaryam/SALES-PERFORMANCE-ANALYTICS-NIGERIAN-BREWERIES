# Project Title
Sales Performance Nigerian Breweries
# Project Overview
Sales performance analysis is critical to understanding profitability and product performance in any Organization. This project uses SQL to analyze profit, country and brand performance for Nigerian Breweries between 2017 and 2019, with a focus on identifying which prodcuts and markets hold a comparative advantage.
# Case Study
Nigerian breweries aim at identifying their best selling product and to achieve this, the marketting Department teamed up with the Data Analytics Department to look into this.The objective is to identify their best selling product and brand, understand sales trends, customer behavior and regional performance across different markets, to improve profitability and market competitiveness. This will be achieved by leveraging a data-driven approach to analysis.For this project, I downloaded my data set from kaggle.
# Tools Used
SQL Server Management Studio
# Data Structure
The data set is a three years sales that consists of 13 columns and 1047 rows.
| Column | Description |
|--------|-------------|
| SalesID | Unique identifier for each sales transaction |
| Sales Rep | Name of the sales representative |
| Email | Sales representative's email address |
| Brand | Product brand sold |
| Plant Cost | Cost of production at the plant |
| Unit Price | Selling price per unit |
| Quantity | Number of units sold |
| Cost | Total cost of the transaction |
| Profit | Profit made on the transaction |
| Country | Country where the sale occurred |
| Region | Region within the country |
| Month | Month of the sale |
| Year | Year of the sale |
